getAllNewsletterSubscribedUsers

This is a functionality for listing all the users subscribed for newsletter.

getAllNewsletterSubscribedUsers
Parameters
req (Object) Request
Name Description
req.query Object req consists of the query string sent through the api route
req.query.subscribed (string | boolean) It provides the value to filter the data
next (any)
Returns
Object: Returns an object containing dataList retrieved from the database, totalItems and currentPage for the pagination support

pagerOpts

Constant that contains the pagination support i.e. page and perpage.

pagerOpts

Type: Object

Properties
page (number) : The current page of the data list.
perpage (number) : The number data in the list for each page.

queryOpts

Query to filter from the database.

queryOpts

Type: Object

Properties
subscribed (boolean) : Filtering value for the data list.

sortOpts

Sorting value to retrieve the data from the database.

sortOpts

Type: Object

Properties
added_on (number) : Determining the sorting by ascending or descending order. Negative refers to descending and vice-versa.
